# Plugin Group Toggle

一个用于 **批量管理 Obsidian 社区插件** 的小插件。  
你可以把已安装插件保存为分组，并通过命令快速 **开启 / 关闭 / 开关** 分组。

## 功能

- 扫描当前已安装的社区插件
- 弹出选择器，勾选插件并保存为分组
- 每个分组自动生成一个命令
- 支持批量开启分组、批量关闭分组
- 分组支持重命名、排序、删除
- 选择器中显示插件当前状态
- 分组操作后可自动刷新当前页面

## 使用方法

### 1. 创建分组
打开命令面板，运行：

- `插件分组开关闭：弹出插件选择器`

在选择器中勾选插件，输入分组名称并保存。

### 2. 开关单个分组
创建分组后，会自动生成命令：

- `插件分组开关闭：开关分组：分组名`

执行时会根据组内每个插件的当前状态逐个切换：

- 当前开启 → 关闭
- 当前关闭 → 开启

### 3. 批量操作分组
命令面板中还提供：

- `插件分组开关闭：关闭插件分组`
- `插件分组开关闭：开启插件分组`

其中：

- **关闭插件分组** 只显示当前处于开启状态的分组
- **开启插件分组** 只显示当前处于关闭状态的分组

## 设置页

插件设置页中可以：

- 打开插件选择器
- 批量开启 / 关闭分组
- 查看并管理已保存分组
- 重命名分组
- 调整分组顺序
- 删除分组
